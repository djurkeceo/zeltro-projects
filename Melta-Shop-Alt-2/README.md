# METAL SHOP d.o.o. — Web sajt

Profesionalni, responzivni web sajt za METAL SHOP d.o.o. iz Subotice — proizvodnja, maloprodaja i veleprodaja metalne robe.

## Tehnologije

- React 19 + TypeScript
- Vite
- React Router

## Pokretanje

```bash
npm install
npm run dev
```

Sajt je dostupan na `http://localhost:5173`.

## Produkcijski build

```bash
npm run build
npm run preview
```
